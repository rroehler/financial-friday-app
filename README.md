# Financial Friday

Financial Friday is a personal financial planning platform designed to help people make better financial decisions through planning, tracking, and actionable recommendations.

Rather than simply showing where money went, Financial Friday helps answer:

> **"What should I do next?"**

---

## Mission

Help people build wealth one decision at a time.

---

## Vision

Create the simplest and most trusted personal financial planning platform.

---

## Current Features

- Secure Firebase Authentication
- Financial Overview Dashboard
- Account Management
- Live Net Worth Tracking
- Firestore Data Storage
- Weekly Financial Trend Chart
- Responsive Mobile-First Interface

# Financial Friday Copy Guide

The language used throughout Financial Friday should be encouraging, positive, and confidence-building.

The application should help users feel successful rather than judged.

---

# Preferred Language

Say:

"You increased your financial freedom by $185."

Instead of:

"You spent $185 less."

---

Say:

"You still have $250 available."

Instead of:

"You have $250 remaining."

---

Say:

"Let's build your plan."

Instead of:

"Edit budget."

---

Say:

"Great progress."

Instead of:

"Good job."

---

# Tone

The application should feel:

- Encouraging
- Calm
- Intelligent
- Friendly
- Professional

Never sarcastic.

Never shame users.

Never use fear-based messaging.

---

# Freedom Mode

Freedom Mode is a planning environment.

Nothing changes until the user chooses to save.

The application should consistently remind users that experimentation is safe.

Examples

"Try different ideas."

"See what happens."

"Nothing is saved until you choose Save."

---

# Tagline

Build financial freedom one Friday at a time.

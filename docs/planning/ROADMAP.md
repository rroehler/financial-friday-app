# ROADMAP.md

# Financial Friday Roadmap

This roadmap provides the long-term vision for Financial Friday.

The roadmap is intentionally flexible and may evolve as the product grows.

---

# Current Phase

## Phase 1 — Foundation

Status: In Progress

### Goals

- Build the core financial dashboard.
- Build the Monthly Plan experience.
- Create a polished and consistent user interface.
- Connect to Firebase.
- Build the foundation for future planning tools.

---

# Phase 2 — Financial Planning

### Objectives

- Editable Monthly Plan
- Freedom Mode
- Import Previous Month
- Planning simulations
- Improved debt planning

---

# Phase 3 — Intelligent Planning

### Objectives

- AI recommendations
- Debt payoff simulator
- Goal planning
- Emergency fund planning
- Savings forecasting

---

# Phase 4 — Automation

### Objectives

- Automatic transaction import
- Spending categorization
- Smart notifications
- Friday Review automation

---

# Phase 5 — Premium Experience

### Objectives

- Shared household planning
- Advanced reporting
- Investment tracking
- Financial coaching
- AI financial advisor

---

# Long-Term Vision

Financial Friday becomes the place where people make financial decisions before spending money.

The application should evolve from budgeting into financial planning and eventually into financial coaching.

# Financial Friday Roadmap

This roadmap provides the long-term vision for Financial Friday.

The roadmap is intentionally flexible and may evolve as the product grows.

---

# Current Phase

## Phase 1 — Foundation

Status: In Progress

### Goals

- Build the core financial dashboard.
- Build the Monthly Plan experience.
- Create a polished and consistent user interface.
- Connect to Firebase.
- Build the foundation for future planning tools.

Financial Friday becomes the place where people make financial decisions before spending money.

The application should evolve from budgeting into financial planning and eventually into financial coaching.

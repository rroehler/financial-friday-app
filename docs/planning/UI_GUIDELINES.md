# Financial Friday UI Guidelines

## Design Philosophy

Financial Friday should feel calm, modern, and trustworthy.

The interface should reduce stress, not create it.

Every screen should help users focus on making better financial decisions.

---

# Design Principles

- Simple over flashy.
- Premium over trendy.
- Information first.
- Consistency everywhere.
- Animations should enhance, never distract.

# Icons

Library

# Animations

Duration
250–350ms

Style
Ease In-Out

Animations should feel similar to Apple Wallet.

Avoid spinning or bouncing effects.

# Layout

# Future Features

New screens should reuse existing components whenever possible.

Avoid creating duplicate button, card, or modal styles.

Consistency is more important than uniqueness.
